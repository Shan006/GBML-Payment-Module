# GBML UI - Payments Module Frontend

React frontend for the GBML Payments Module (JRC-20).

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

## Configuration

The frontend connects to the backend API at `http://localhost:3000/gbml` by default.

To change this, update the `API_BASE_URL` constant in the component files.

## Features

- Enable Payments Module (deploy JRC-20 token)
- Send Payments (JRC-20 transfers)
- View transaction results
